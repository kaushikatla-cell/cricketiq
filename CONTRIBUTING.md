# Contributing to CricketIQ

Thank you for helping improve CricketIQ. This project is intentionally **JavaScript-only** (no TypeScript) and **localStorage-first** for the MVP.

## How to contribute

1. Fork the repository and create a branch from `main`.
2. Install dependencies: `npm install`
3. Run checks before opening a PR:
   - `npm run lint`
   - `npm run build`
4. Open a pull request with a clear description of the change and any user-facing impact.

## Guidelines

- Prefer small, focused PRs over large refactors unless discussed first.
- Match existing patterns for components, Tailwind classes, and `lib/` modules.
- Do not commit secrets, API keys, or personal data exports.

## Questions

Open an issue if you are unsure about scope or design direction.
